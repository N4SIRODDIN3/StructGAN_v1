codes for structGAN evaluation
folder "images_eval" has the target and output images

**use codes in order:
1_rename_images, to keep all images with .png format
2_subdiv_images, to subdivide target and output images
3_IoUdet, to evaluate performance by IoU
4_confusion_matrix, to evaluate performance by confusion matrix
5_SWratio, to evaluate performance by SWratio
6_meanstd, to obtain mean and standard value
